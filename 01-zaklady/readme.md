# Obsah

- **Proměnné a datové typy**: var, val, typová inference,převody typů  
- **Vstup od uživatele**: readln(), argumenty příkazové řádky (args)  
- **Podmínky (if - else)**: podmíněné větvení, kontrola délky stringu  
- **Příkaz when**: ekvivalent switch, vícenásobné větvení  
- **Cyklus**: for (rozsahy, pole), while, do-while  
- **Kolekce**: listOf, mutableListOf, ArrayList, arrayOf, setOf, mapOf  
- **Nullable a filtrace**: práce s null hodnotami, filterNotNull  
- **Datové třídy a objekty**: data class, definice tříd, metody
